 Project Overview

This project performs detailed Exploratory Data Analysis (EDA) on the Netflix dataset to uncover insights about:

Content distribution

Genre trends

Release patterns over the years

Country-wise content availability

Ratings and audience categories

Movies vs TV Shows comparison

The dataset contains information such as title, director, cast, release year, rating, duration, country, and type.

Objectives

Understand the structure of Netflix’s content library

Identify trends in movies vs TV shows

Analyze popular genres/categories

Study content addition over years

Explore country-based content production

Visualize rating classifications

 Dataset Description
